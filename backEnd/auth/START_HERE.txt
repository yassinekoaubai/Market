═══════════════════════════════════════════════════════════════════════════════
                              📊 ENDPOINTS API
═══════════════════════════════════════════════════════════════════════════════

POST /auth/register
  • Créer un nouveau compte
  • Body: email, password, confirmPassword, name, phone

POST /auth/login
  • Connexion utilisateur
  • Body: email, password

POST /auth/refresh-token
  • Renouveler access token
  • Body: refreshToken

POST /auth/logout
  • Déconnexion (revoke token)
  • Headers: Authorization: Bearer <token>

POST /auth/logout-all
  • Déconnexion tous appareils
  • Headers: Authorization: Bearer <token>

GET /health
  • Vérifier l'état du serveur

                              ⚠️  IMPORTANT
═══════════════════════════════════════════════════════════════════════════════

1. ⚠️  DATABASE_URL - Obligatoire dans .env
   Format: postgresql://user:password@host:port/database

2. ⚠️  JWT_SECRET - Changez en production!
   Générez une clé forte: openssl rand -base64 32

3. ⚠️  CORS_ORIGIN - Configurer selon vos domaines
   Défaut: http://localhost:3000,http://localhost:3001

4. ⚠️  .env - Jamais committer dans Git!
   C'est dans .gitignore car contient des secrets
